## como usar
* clone o repositorio:
`git clone https://github.com/lazaro-pontes/botLikeInsta`
* nevegue ate a pasta do repositorio:
`cd botLikeInsta/`
* instale as dependencias do Node:
`npm install`
* execute o bot:
`npm run start`
