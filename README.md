## como usar
* clone o repositorio:
`git clone https://github.com/lazaro-pontes/botLikeInsta`
* nevegue ate a pasta do repositorio:
`cd botLikeInsta/`
* instale as dependencias do Node:
`npm install`
* crie um arquivo `.env` para armazenar as variaveis de ambiente `EMAIL` e `PASSWORD`
* execute o bot:
`npm run start`
